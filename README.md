# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe game** built using **HTML, CSS, and JavaScript**.
This project allows a user to play against the computer with a clean user interface and basic AI logic.

## ✨ Features

* Player vs Computer gameplay
* Computer AI move system
* Winner highlight with green color
* Celebration popup when someone wins 🎉
* Restart button to play again
* Score tracking
* Attractive UI with glow effects

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript

## 🎯 How to Play

1. Click on any empty cell to place your **X**.
2. The computer will automatically place **O**.
3. The first player to align **3 symbols in a row, column, or diagonal wins the game**.
